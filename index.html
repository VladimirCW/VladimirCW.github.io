Instal Java

  1. Go to the link and download latest LTS package of JDK.
  2. After downloading install it.
  3. Check that java was properly installed
  4. Open any console (for example CMD - press Win+R -> type "cmd" -> press Enter)
  5. In console type "java -version
  6. In console type "javac -version"

If you see an error 'javac' is not recognized as an internal or external command, operable program or batch file. you need to set environment variable for java

Set up Java environment variables

  1. Start
  2. This PC
  3. Click mouse right button -> "Properties"
  4. Advanced system settings
  5. Advanced -> Environment variables
  6. Add new system variable JAVA_HOME = path to installed java
  7. Save it
  8. Select system "Path" variable -> click Edit
  9. Click New -> type %JAVA_HOME\bin%
  10. Save

Reopen console and check one more time java -version javac -version You should see similar message


Install maven

  1. Go to link
  2. Download binary zip archive
  3. Extract archive into C root folder
  4. Set system variable M2 as 'C:\apache-maven-{version}\bin'
  5. Add %M2% variable to Path system variable
  6. Check that maven was installed mvn -version
